Create Firestore Index for Coaches Collection

This page will redirect you to the Firebase Console to create the required index for the coaches collection.

The error you're seeing is because Firestore requires a composite index for the query you're trying to execute.

Note: You'll need to be logged into your Firebase account to create the index.

What is happening?

Your app is trying to query the coaches collection with multiple ordering or filtering conditions, which requires a composite index.

This specifically looks like a query that's filtering by isVerified and ordering by name.

After clicking the button above and creating the index, you'll need to wait a few minutes for the index to be created and deployed.
